## Getting Started

> Install virtualenv to proceed
> While being in Project root dir (the same dir this README exists in)

```
virtualenv venv --python=python3
```

```
. ./venv/bin/activate
```

```
pip install -r requirements.txt
```
### Prerequisites

* Python3
* Postgresql
* virtualenv
